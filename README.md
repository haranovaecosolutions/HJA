# Hemil Joshi & Associates - Bright Premium React Website

Premium React + Vite website for Hemil Joshi & Associates, with bright premium colours, smaller profile photo presentation, animated hero section, service cards, managing partner section, contact form and Vercel-ready serverless API.

## Run locally
```bash
npm install
npm run dev
```

## Deploy on Vercel
1. Push this folder to GitHub.
2. Open Vercel > Add New Project > Import Git Repository.
3. Framework preset: Vite.
4. Build command: `npm run build`.
5. Output directory: `dist`.
6. Deploy.

## Images
- `public/hemil-joshi-profile.jpg` is extracted from the supplied profile file and displayed in a smaller premium frame.
- `public/ca-logo.png` is extracted from the supplied firm masthead. For production, replace it with the official ICAI CA India logo downloaded by the member and use it only as per ICAI logo guidelines.
- `public/firm-masthead.jpg` is extracted from the supplied firm profile and used in the contact section.
